Diferença entre tags de link e download

A tag de link pode ser feita com link interno e externo, vamos utilizar os dois exemplos, nos dois nós utilizamos a tag <a>.
Porém para o link interno nos utilizamos o atributo "href" e o atributo target como tendo o valor = "_self" além disso, colocamos o atributo rel com o valor em relação a ordem das páginas, ou seja "prev" para a anterior e "next" para a próxima.
